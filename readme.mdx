It is a telegram bot, you can send it a list of songs and it will create a playlist in spotify and add all songs in it. 
Stack:
- Iogram
- Pytube
- Spotipy
- Re

You can try to use it by yourself, however, you need to install all dependencies from the file requirements.txt and create a file called ".env" in the main catalog of directory and add in it "SPOTIPY_CLIENT_ID = secret
SPOTIPY_CLIENT_SECRET = secret
SPOTIPY_REDIRECT_URI = secret
USER_NAME = secret
TELEGRAM_TOKEN_BOT = secret
" with your own values instad of secret
